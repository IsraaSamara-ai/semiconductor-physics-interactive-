وصف المستودع / Repository Description
العربية:
فيزياء أشباه الموصلات التفاعلية — تطبيق تعليمي تفاعلي مُصمَّم لشرح مفاهيم فيزياء الإلكترونيات بشكل مبسط وبصري. يغطي التطبيق: الفجوة الطاقية، أشباه الموصلات من النوع N والنوع P، الثنائي البلوري (PN Junction)، الانحياز الأمامي والعكسي، مقاومة الثنائي البلوري، واستخدامه كمقوِّم للتيار المتردد. يتميز برسوم متحركة تفاعلية تُوضّح حركة الإلكترونات والفجوات، ومنحنيات بيانية خطوة بخطوة، وأدوات تحكّم تتيح للطالب تغيير المادة والقيم ورؤية النتائج فوراً. مُعدّ لطلاب الصف الثاني عشر الذين لا يمتلكون خلفية مسبقة في الموضوع.

English:
Interactive Semiconductor Physics — An interactive educational application designed to explain electronics physics concepts visually and simply. Covers: band gap, N-type and P-type semiconductors, PN junction diode, forward and reverse bias, diode resistance, and AC rectification. Features interactive animations showing electron and hole movement, step-by-step curves, and controls for changing materials and values with instant visual feedback. Designed for 12th-grade students with no prior background.

| القسم | المحتوى التفاعلي |
|-------|-----------------|
| **المقدمة** | نظرة عامة على المفاهيم وكيفية الاستخدام |
| **المواد وشبه الموصلات** | رسوم متحركة لبلورة السليكون + منزلق حرارة يُظهر تكسّر الروابط وتكوّن الإلكترونات الحرة والفجوات |
| **الفجوة الطاقية** | مخطط نطاقات الطاقة التفاعلي (Plotly) مع تبديل بين Si و Ge |
| **النوع N و P** | رسوم متحركة تُظهر الإشابة مع اختيار نوع الشائبة ورؤية الناقلات الأغلبية والأقلية |
| **الثنائي البلوري** | رسوم متحركة لوصلة PN تُظهر الانتشار ومنطقة الاستنزاف والمجال الداخلي |
| **الانحياز الأمامي** | دارة متحركة كاملة (بطارية+مقاومة+ثنائي+مصباح) مع منزلق جهد يُظهر حركة التيار وتأثير حاجز الجهد |
| **الانحياز العكسي** | نفس الدارة مع انعكاس البطارية، يُظهر اتساع منطقة الاستنزاف وانعدام التيار |
| **مقاومة الثنائي** | منحنى I-V تفاعلي (Plotly) مع نقطة متحركة تعرض المقاومة المحسوبة عند أي جهد |
| **مقوّم التيار المتردد** | رسوم متحركة تُظهر الإشارة الداخلة والدارة والإشارة الخارجة مع منحنيات Plotly |
